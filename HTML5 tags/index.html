Video Tags

HTML5 has several new useful tags. The first we will look at is the video tag. The video tag allows you to embed a video into your webpage by specifying the format and source of the video. The tag also allows you to support several video types. The browser will go through each video format until it finds one that works with the current device. If none of the formats are compatable, then you can have the browser display an error message. Any text between the begining and ending video tags will be displayed in browsers that do not support the element if you should chose not use an error message.

There are several opptional attributes that can be used with the video tag.

Attribute Value Description
autoplay autoplay Specifies that the video will start playing as soon as it is ready
controls controls Specifies that video controls should be displayed (such as a play/pause button etc).
height pixels Sets the height of the video player
loop loop Specifies that the video will start over again, every time it is finished
muted muted Specifies that the audio output of the video should be muted
poster URL Specifies an image to be shown while the video is downloading, or until the user hits the play button
preload auto
metadata
none
 Specifies if and how the author thinks the video should be loaded when the page loads
src URL Specifies the URL of the video file
width pixels Sets the width of the video player

Code Example


<video width="100%" height="auto" controls>
    <source src="video/creepySquirrel.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Video of a clueless bunny and a creepy squirrel.

Audio Tag

The audio tag works the same way as the video tag. See the code below. 


<audio controls="">
    <source src="audio/OceanWaves.mp3" type="audio/mpeg">
    Your browser does not support this audio file.
</audio>

Canvas Tag

The canvas tag allows you to draw graphics on a webpage with scripting (usually JavaScript). This element is only a container for graphics. A script must be used to actually draw the graphics. Canvas has several methods for drawing paths, boxes, circles, text, and adding images. Canvas can be animated, interactive, and can also be used in games.


Code Example

            var canvas = document.getElementById('canvas');
            var ctx = canvas.getContext('2d');
		var ntd = canvas.getContext('2d');
