Variables

Variables store data values. They act very similar to the variables in Algebra. They act as containers that hold values. Variables are always defined with the keyword var followed by the vairable name. Variable names can contain letters, digits, underscores, and dollar signs and are case sensative. For example x and X are two different variables. Variables use the assingment opporator "=" to assign a value to the variable.

Functions

JavaScript uses Functions to perform certain tasks. A function is a block of code that is executed when something calls it. A function always starts with the keyword function followed by the name of the function and then with parentheses. The function name follows the same criteria as a variable name. They can contain letters, digits, underscores, and dollar signs and are case sensative. The parentheses after the function name can contain multiple parameters seperated by commas. These act as local vairables. After the function and parameters, the code to be executed is placed within curly brackets {}.

Conditional Statements

Conditional statments allow you to write code that can perform different actions for different scenerios. There are four different types of Conditional Statements.
-if statements are used to execute a block of code if a specified condition is true.
-else statements are used to execute a block of code if the same condition is false.
-else if statements are used to execute a block of code if the first condition is false but the second condition is true.
-switch statements are used to execute many different blocks of code.

Loops

Loops run the same code over and over again each time resulting in different values. These are especially useful when working with Arrays.

According to W3Schools.com, there are several different kinds of loops.

for - loops through a block of code a number of times
for/in - loops through the properties of an object
while - loops through a block of code while a specified condition is true
do/while - also loops through a block of code while a specified condition is true

For Loops

For-Loops are comprised of three statements. The first statement is code that is executed before the loop or the code block starts. The second statement is in charge of defining the condition for running the code block or loop. The third statement starts each time the code block or loop has been executed.

Example

for (statement 1; statement 2; statement 3) {
code block to be executed
}
