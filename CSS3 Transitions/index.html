Nodes

Everything in an HTML DOM is a node. There are several different Node types such as:

    - Document node
    - Element nodes
    - Attribute nodes
    - Text inside HTML elements are text nodes
    - Comments nodes

Each node can have a child node. Nodes on the same level are called sibling nodes.

We can then execute the following code.


		var newPara = document.createElement("p");
		var node = document.createTextNode("This is a new paragraph.");
		newPara.appendChild(node);

		var element = document.getElementById("main");
		var child = document.getElementById("first");
