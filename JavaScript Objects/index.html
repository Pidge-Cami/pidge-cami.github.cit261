JavaScript Objects

Javascript Objects are variables that can store many values. A wizard, for example, is an object and it has properties.

Properties can be things such as name, school, age, or gender and are the named values in JavaScript objects. The values are written as name:value pairs. This means that the name of the variable comes first followed by the value and are seperated by a colon.
For example; name:"Harry Potter".
These pairs together are called Properties. All wizards have the same properties, but the values may differ. All wizards also have methods, but they may be performed at different times. Methods are actions that can be performed on objects and are stored in properties as function definitions.

Creating Objects

To create an object in JavaScript, it must be delcared. The keyword var is used to declare an object, followed by the object name.

var objectname

Another way would be to use an "object type" and create many objects of one type. This is done by using a constructor function:

function wizard(name, school, age, gender) {
  this.name = name;
  this.school = school;
  this.age = age;
  this.gender = gender;
  }

You can create an object using the the keyword new using that constructor: 

var objectname = new FunctionName
var Ron = new wizard())
