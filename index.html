The DOM

The DOM or the Document Object Model is an object oriented model of a web page's logical structure.

The DOM of an HTML document can be shown in a tree structure. The tree is made up of HTML elements that are interrelated nodes.

Nodes

Everything in an HTML DOM is a node. There are several different Node types such as:

    - Document node
    - Element nodes
    - Attribute nodes
    - Text inside HTML elements are text nodes
    - Comments nodes

Each node can have a child node. Nodes on the same level are called sibling nodes.
